## 1. 系统顶部导航栏

> **章节导读**：本节介绍互联网边界防火墙控制台的顶部导航栏结构。导航栏是用户在不同功能模块间切换的主要入口，包含四个核心功能标签：**概览**、**防火墙开关**、**访问控制**、**日志**。通过导航栏，用户可以快速访问防火墙的各个管理页面，完成从整体监控到具体配置的全部操作。

| **日志** | 查看威胁日志和流量日志 |

## 2. 系统概览页面

> **章节导读**：本节介绍系统概览页面的所有内容模块。概览页面是防火墙的"仪表盘"，集中展示防火墙的运行状态和安全数据。页面分为五个主要区域：**安全防护**显示累计防护天数、资产数量和流量统计；**安全监控**展示威胁总数和趋势图表；**流量监控**以可视化图表形式呈现总流量、上行流量、下行流量的时间趋势；**资源信息**显示防火墙实例的地域、服务状态、可防护 IP 数和带宽值；**付费信息**展示创建时间、过期时间、自动续费状态和计费方式。管理员可以通过此页面快速掌握防火墙的整体运行状况。

## 4. 访问控制页面

> **章节导读**：本节介绍访问控制页面的功能和字段说明。访问控制页面是配置防火墙策略的核心界面，用于定义哪些流量可以通过防火墙。页面左侧导航包含**IPv4 策略**、**IPv6 策略**和**地址簿管理**三个模块。顶部提供**出向**和**入向**策略方向切换，分别控制内到外和外到内的流量。操作按钮栏支持新建、启用、禁用策略及批量操作。策略列表以表格形式展示所有已配置的策略，包含优先级、策略名称、访问源、目的、协议类型、端口、动作、命中次数、创建时间等关键信息。分页控件用于浏览大量策略记录。

> <span style="color: red; font-weight: bold;">若未配置任何访问控制策略，云防火墙将默认放行所有流量。您可通过创建互联网边界访问控制策略，对公网资产访问互联网的流量实现精细化管控。</span>

### 4.1 左侧导航

| 导航项 | 字段解释 |
|--------|----------|
| **IPv4 策略** | 管理 IPv4 协议的访问控制策略 |
| **IPv6 策略** | 管理 IPv6 协议的访问控制策略 |
| **地址簿管理** | 管理 IP 地址组（地址簿），便于策略引用 |

### 4.2 策略方向切换

| 选项 | 字段解释 |

### 4.3 操作按钮栏

| 按钮名称 | 字段解释 |
|----------|----------|
| **新建策略** | 创建新的访问控制策略 |
| **启用** | 启用已选中的策略（使策略生效） |
| **禁用** | 禁用已选中的策略（使策略失效） |
| **删除** | 删除已选中的策略 |
| **搜索框** | 输入关键词查询策略 |
| **刷新** | 刷新策略列表 |
| **文档** | 查看帮助文档 |

### 4.4 策略列表

| 字段名称 | 字段解释 |
|----------|----------|
| **复选框** | 用于选择策略进行批量操作 |

| **策略名称** | 策略的自定义名称标识 |
| **访问源** | 发起访问请求的源 IP 地址或地址簿 |
| **目的** | 访问的目标 IP 地址、地址簿或域名 |

## 5. 策略配置页面

> **章节导读**：本节介绍新建/编辑访问控制策略时的配置页面。策略配置页面是定义防火墙规则的核心界面，包含多个配置区域。**策略配置区域**需要填写访问源（源 IP 或地址簿）、目的（目标 IP、地址簿或域名）、协议类型（TCP/UDP/ICMP/ANY）、端口（0-65535）和动作（放行/拒绝）。**优先级/有效期区域**用于设置策略的优先级（最高/最低/自定义）、有效期（始终/单次/时间段）和启用状态（ON/OFF）。**其它设置**包含策略名称字段，用于标识策略用途。底部提供取消和确认按钮，完成策略的创建或编辑操作。

### 5.1 策略配置区域

#### 访问源

| 字段名称 | 字段解释 |
|----------|----------|
| **IP 地址** | 直接输入源 IP 地址，支持 CIDR 格式（如 192.168.1.0/24） |
| **地址簿** | 选择已创建的地址簿作为源地址 |
| **输入框提示** | 请输入单个 IPv4 地址或地址段，如：100.100.100.100/32；多个 IP 用英文逗号分隔 |

#### 目的

| 字段名称 | 字段解释 |
|----------|----------|
| **IP 地址** | 直接输入目标 IP 地址 |
| **地址簿** | 选择已创建的地址簿作为目标地址 |
| **域名** | 直接输入目标域名（如：www.example.com） |
| **输入框提示** | 支持 IPv4 地址段、地址范围、全部地址（0.0.0.0/0） |

#### 协议类型

| 选项 | 字段解释 |
|------|----------|
| **TCP** | 传输控制协议，面向连接的可靠传输协议 |
| **UDP** | 用户数据报协议，无连接的快速传输协议 |
| **ICMP** | 互联网控制消息协议，用于 ping 等网络诊断 |
| **ANY** | 任意协议，匹配所有协议类型 |

#### 端口

| 字段名称 | 字段解释 |
|----------|----------|
| **端口** | 指定具体的目的端口号 |
| **端口格式说明** | 单个端口：80；端口范围：22/25 表示 22、23、24、25；多个端口用英文逗号分隔 |
| **端口范围** | 有效端口值：0-65535 |

#### 动作

| 选项 | 字段解释 |
|------|----------|
| **放行** | 允许匹配的流量通过防火墙 |
| **拒绝** | 阻止匹配的流量通过防火墙 |

### 5.2 优先级/有效期区域

#### 优先级

| 选项 | 字段解释 |
|------|----------|
| **最高** | 策略优先级最高，最先被匹配 |
| **最低** | 策略优先级最低，最后被匹配 |
| **自定义** | 手动输入优先级数值 |

#### 有效期

| 选项 | 字段解释 |
|------|----------|
| **始终** | 策略永久有效 |
| **单次** | 策略仅生效一次 |
| **时间段** | 策略在指定时间段内有效 |

#### 启用状态

| 选项 | 字段解释 |
|------|----------|
| **ON** | 策略创建后立即启用 |
| **OFF** | 策略创建后暂不启用 |

### 5.3 其它设置

| 字段名称 | 字段解释 |
|----------|----------|
| **策略名称** | 为策略设置一个易于识别的名称（必填） |

## 6. 地址簿管理

> **章节导读**：本节介绍地址簿管理功能。地址簿是 IP 地址的逻辑分组工具，用于简化策略配置和管理。通过地址簿，管理员可以将多个 IP 地址或网段归为一组，在配置策略时直接引用地址簿，而无需逐个输入 IP 地址，大大提高配置效率。新建地址簿表单包含两个必填字段：**IP 地址**字段支持多种输入格式（单个 IP、CIDR 网段、地址范围、多个地址），**地址簿名称**字段用于设置易于识别的分组名称。底部提供取消和确认按钮完成操作。

### 6.1 新建地址簿表单

| 字段名称 | 必填 | 字段解释 |
|----------|------|----------|
| **IP 地址** | ✓ | 输入要加入地址簿的 IP 地址或地址段 |
| **地址簿名称** | ✓ | 为地址簿设置一个易于识别的名称 |

> <span style="color: red; font-weight: bold;">所有已开启防护的 EIP，将默认加入名为 local 的地址簿中。</span>

### 6.2 IP 地址输入格式说明

| 格式类型 | 示例 | 说明 |
|----------|------|------|
| **单个 IP** | `100.100.100.100` | 指定单个 IP 地址 |
| **CIDR 网段** | `100.100.100.100/32` | 使用 CIDR 表示法的网段 |
| **地址范围** | `100.100.100.100-100.100.100.200` | 连续的 IP 地址范围 |
| **多个地址** | `192.168.1.1,192.168.1.2` | 用英文逗号分隔多个 IP |
| **全部地址** | `0.0.0.0/0` | 表示任意 IP 地址（注意不要重复输入） |

## 7. 日志管理页面

> **章节导读**：本节介绍日志管理页面的功能和字段说明。日志管理是防火墙安全审计和故障排查的重要工具，分为**威胁日志**和**流量日志**两种类型。威胁日志记录防火墙检测到的各种安全威胁和攻击事件，包括源 IP、目的 IP、威胁名称、威胁类型、威胁等级等详细信息。查询条件区域提供时间选择、关键词搜索、刷新等功能，帮助管理员快速定位特定时间段或特定类型的日志记录。日志列表以表格形式展示，支持分页浏览。通过分析日志，管理员可以了解网络攻击趋势、优化策略配置、追溯安全事件。

### 7.1 日志类型导航

| 日志类型 | 字段解释 |
|----------|----------|
| **威胁日志** | 记录防火墙检测到的安全威胁和攻击事件 |
| **流量日志** | 记录通过防火墙的网络流量信息 |

### 7.2 查询条件区域

| 字段名称 | 字段解释 |
|----------|----------|
| **时间选择** | 下拉选择预设时间范围（如：1 小时） |
| **时间范围** | 显示当前查询的起始和结束时间（如：2026-04-17 17:17:08 - 2026-04-17 18:17:08） |
| **搜索框** | 输入完整内容关键词进行全文检索 |
| **刷新** | 重新加载日志数据 |
| **文档** | 查看日志相关帮助文档 |

### 7.3 威胁日志

| 字段名称 | 字段解释 |
|----------|----------|
| **源 IP** | 发起威胁请求的源 IP 地址 |
| **目的 IP** | 威胁请求的目标 IP 地址 |
| **威胁名称** | 检测到的威胁的具体名称标识 |
| **源端口** | 发起请求的源端口号 |
| **目的端口** | 请求访问的目标端口号 |
| **威胁类型** | 威胁的大类分类（如：漏洞利用、恶意软件） |
| **威胁子类型** | 威胁的详细分类（如：SQL 注入、XSS 攻击） |
| **应用/协议** | 关联的应用程序或网络协议 |
| **响应行为** | 防火墙对该威胁采取的响应动作 |
| **威胁等级** | 威胁的严重程度级别（如：低、中、高、严重） |
